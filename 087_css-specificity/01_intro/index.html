Specificity Calculator

  1. Specificity determines, which CSS rule is applied by the browsers.
  2. Specificity is usually the reason why your CSS-rules donâ€™t apply to some elements, although you think they should.
  3. Every selector has its place in the specificity hierarchy.
  4. If two selectors apply to the same element, the one with higher specificity wins.
  5. There are four distinct categories which define the specificity level of a given selector:
    1. inline styles
    2. IDs
    3. classes+attributes
    4. elements
  6. When selectors have an equal specificity value
    • the latest rule is the one that counts.
  7. When selectors have an unequal specificity value, the more specific rule is the one that counts.
  8. The universal selector and inherited selectors have a specificity of 0, 0, 0, 0.
